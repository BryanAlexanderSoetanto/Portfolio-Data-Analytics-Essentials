**Pengantar Data**

Definisi data: “Facts and statistics collected together for reference or analysis.” (Oxford Languages).
Data adalah fakta/statistik yang dikumpulkan untuk referensi atau analisis.

DIKW Pyramid: (Data, Information, Knowledge, Wisdom)

**Apa itu Data Analytics?**
Analisis data (biasanya dalam jumlah besar) menggunakan matematika, statistik, dan software komputer.
Istilah data analytics dan data analysis sering digunakan secara bergantian.

**Proses Analisis Data**
Menjelaskan tahapan-tahapan yang dilakukan dalam analisis data (meski detail tidak dijelaskan panjang di slide).

**Jenis-Jenis Analisis Data**
Deskriptif, diagnostik, prediktif, preskriptif).
